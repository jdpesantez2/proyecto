Robot LetsCode

El robot de Let’s Code es un robot educativo que se programa mediante bloques visuales. Este tipo de programación permite controlar hardware real de forma sencilla, entendiendo la lógica sin necesidad de escribir código complejo.

La programación por bloques está diseñada para interactuar con sensores, motores y componentes físicos, facilitando el aprendizaje de la robótica y la automatización.

A continuación, conoceremos las partes que complementan el robot.

Partes del Robot

Speaker / Altavoz
Dispositivo que emite sonido. Se utiliza para reproducir alertas, señales acústicas o efectos sonoros generados por el sistema del robot.
IR Receiver / Receptor IR
Componente que recibe señales infrarrojas, normalmente desde un control remoto. Permite controlar el robot de forma inalámbrica mediante comandos.
Ultrasonic Distance Sensor
Sensor que mide la distancia a los objetos usando ondas ultrasónicas. Se usa para detectar obstáculos y evitar colisiones.
2x Infrared Reflection Sensors
Sensores que detectan superficies claras u oscuras mediante reflexión infrarroja. Se usan comúnmente para seguir líneas o detectar bordes.
ESP32 Controller Module
Es el microcontrolador principal del sistema. Procesa la información de los sensores y controla los motores, luces y demás componentes del robot.
I2C Interface / Interfaz I2C
Puerto de comunicación que permite conectar otros módulos o sensores usando el protocolo I2C, facilitando la expansión del sistema.

Lo que el Robot puede hacer

Cuando programamos con bloques, el robot puede realizar varias tareas: cambiar de luces, seguir una línea, evitar chocar con objetos y seguir una fuente de luz.
